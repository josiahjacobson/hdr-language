High Dynamic Range is a concept used for modern audio-visual systems, such as increased brightness range (eg. 0-4,000 nits), color (eg. BT2020) as well as audio (eg. 24-bit 96Khz).

This project applies that concept to language, for everyday words as well as Scripture. 

Markup will be supported, for purposes eg. inline processing, visual notations as well as other suitable encoding for procedural (non-AI) or machine learning (AI).

In this manner it is proposed the primary Word and word can be compressed and communicated with human readability (to a certain degree) orders of magnitude more towards people and digital systems.
